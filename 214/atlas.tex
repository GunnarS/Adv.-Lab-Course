\section{Einleitung}
In diesem Praktikumsversuch soll die Physik des ATLAS-Experimentes kennengelernt und untersucht werden. So wird sich einleitend anhand von \emph{event displays} mit den zu Grunde liegenden Prozessen und dem ATLAS vertraut gemacht. Anschließend wird die Messung der Elektronenenergie anhand der $Z^0$-Masse kalibriert, woraufhin dann entweder eine Bestimmung der W-Boson-Masse oder aber die Suche nach neuer Physik anhand von simulierten ATLAS-Daten erfolgt.

\section{Theoretische Grundlagen}
\subsection{Das Standardmodell}
Das Standardmodell der Teilchenphysik ist eine relativistische Quantenfeldtheorie und beschreibt mittels der drei Eichgruppen SU(3) $\times$ SU(2) $\times$ U(1) die Elementarteilchen und deren Wechselwirkungen miteinander.

\subsubsection{Teilchen}
Das Standardmodell kennt zwei Sorten von Teilchen, die elementar oder zusammengesetzt sein können: Fermionen mit halbzahligem Spin und Bosonen mit ganzzahligem. Im Folgenden sollen nur Elementarteilchen betrachtet werden. 

Die Fermionen sind in Tabelle \ref{tab:ferm} aufgeführt. Man unterscheidet zwischen den starkwechselwirkenden Quarks aus denen die Baryonen und Mesonen aufgebaut sind und Leptonen. Beide können in drei Generationen oder Familien angeordnet werden (siehe Tabelle \ref{tab:ferm}). Neben der aufgeführten Eigenschaft der elektrischen Ladung haben im Standardmodell bis auf die Neutrinos alle Quarks und Leptonen eine charakteristische Masse, die von wenigen $\unit{MeV}$ für Up- und Down-Quark bis hin zu über $\unit[100]{GeV}$ für das Top-Quark reicht.

\subsubsection{Wechselwirkungen}
Das Standardmodell kennt drei fundamentale Wechselwirkungen: die elektromagnetische, die schwache und die starke Wechselwirkung. 

Die elektromagnetische Wechselwirkung wird mittels einer abelschen U(1) Eichgruppe beschrieben, die Eichbosonen (Photonen) können also nicht aneinander koppeln. Da Photonen masselos sind, hat die elektromagnetische Wechselwirkung eine unendliche Reichweite.

Die über die Eichgruppe SU(2) beschriebene schwache Wechselwirkung lässt sich mit der elektromagnetischen zur sogenannten elektroschwachen Wechselwirkung SU(2) $\times$ U(1) vereinheitlichen. Auf Grund ihrer massiven Eichbosonen ist die Reichweite der schwachen Wechselwirkung sehr klein, jedoch koppeln diese an alle Teilchen.

Die starke Wechselwirkung wird von Gluonen vermittelt, welche die Ladung der starken Wechselwirkung, die Farbladung, selbst tragen, Selbstwechselwirkung ist also möglich. Beschrieben wird die Theorie der starken Wechselwirkung durch die nicht-abelsche Eichgruppe SU(3). Gluonen sind zwar masselos, haben aber auf Grund der Selbstwechselwirkung nur eine geringe Reichweite. Es existieren nur gebundene, farblose Zustände, freie Quarks oder Gluonen gibt es nicht (\emph{confinement}). Bei sehr kleinen Abständen, also hohen Übertragsimpulsen, sind Quarks "`asymptotisch frei"'.

\subsection{Neue Physik}
\label{cha:newph}
Die über das Standardmodell hinausgehenden theoretischen Überlegungen werden auch als "`neue Physik"' bezeichnet. Man erhofft sich vom LHC (auch über ATLAS) u.\ a.\ hierzu Hinweise. 
Die hier relevanten Kandidaten für neue Physik sind:
\begin{enumerate}
\item{Das \textbf{Higgs-Boson} ist zwar Teil des Standardmodells und wird dort benötigt, um den schweren Eichbosonen sowie den übrigen Teilchen ihre Masse zu geben, wurde bisher aber noch nicht experimentell nachgewiesen, weshalb es hier unter "`neuer Physik"' aufgeführt wird. In der einfachsten Theorie ist das Higgs-Boson ein reelles skalares Feld, hat also Spin 0 und würde sich über einen Peak in der Verteilung der invarianten Masse zeigen.}
\item{In der Theorie der \textbf{Supersymmetrie} wird zu jedem Standardmodell-Teilchen ein  Partnerteilchen postuliert und zwar zu Fermionen bosonische und zu Bosonen fermionische supersymmetrische Partner. Die Massen der SUSY-Teilchen sind nicht bekannt, sie sollten jedoch deutlich höher liegen als die ihrer Partnerteilchen, da man sonst beispielsweise den SUSY-Partner zum Elektron bereits gefunden haben müsste. Auch hier gibt es mehrere zur Auswahl stehende Theorien, wobei wir uns hier auf eine Theorie mit Neutralino als LSP (lightest supersymmetric particle) beschränken. Das Neutralino müsste sich im Experiment durch fehlenden Transversalimpuls kennzeichnen.}
\item{Eine weitere Möglichkeit für neue Physik sind \textbf{schwere Quarks}. So ist es durchaus denkbar, dass es noch eine weitere, vierte Leptonen- und Quark-Generation gibt. Aus der Breite des $Z^0$-Peaks erhält man jedoch als obere Grenze für die Anzahl der leichten Neutrinos drei, gäbe es also eine vierte Familie, so müssten die Teilchen dieser  deutlich schwerer sein als die bisherigen. Die Quarks dieser Familie wären also schwer und würden sich vor allem durch eine hohe Anzahl hadronischer Jets auszeichnen.}
\item{In verschiedenen Theorien werden \textbf{weitere schwere Eichbosonen $Z'$} vorhergesagt. Hier sollen die Daten beispielsweise darauf untersucht werden, ob es ein weiteres schweres Eichboson mit ähnlichen Eigenschaften wie das $Z^0$ gibt, das aber deutlich schwerer wäre als dieses. Es soll ebenfalls an Quarks und Leptonen gleichermaßen koppeln und würde in ein Leptonenpaar oder ein Quarkpaar zerfallen. Es kann nicht nur einzeln, sondern auch paarweise oder zusammen mit einem $Z^0$ produziert werden und würde mit letzteren beiden Möglichkeiten zu einem Vier-Leptonen-Endzustand beitragen, was für die Auswertung der Daten interessant ist.}
\end{enumerate}

\subsection{Kinematik}
\subsubsection{Zwei-Körper-Zerfall}
Der Zwei-Körper-Zerfall beschreibt den Zerfall eines Teilchens $A$ in zwei andere Teilchen ($A\ \rightarrow\ B\ C$), welche im Ruhesystem von $A$ genau entgegengesetzte Impulse mit Betrag
\begin{align*}
|\vec{p}_B| = |\vec{p}_C| = \frac{\left[ \left( M^2_A - \left( M_B + M_C \right)^2 \right) \left( M^2_A - \left( M_B - M_C \right)^2 \right) \right]^{\frac{1}{2}}}{2 M_A}
\end{align*}
besitzen. Können die Massen $M_B$ und $M_C$ im Vergleich zu $M_A$ vernachlässigt werden, erhält man $|\vec{p}_{B,C}| = \frac{1}{2} M_A$.

Aus der Vier-Impuls-Erhaltung $p_A = p_B + p_C$ ergibt sich für die sogenannte invariante Masse $M^2_A = p^2_A$:
\begin{align*}
M^2_A = (p_B + p_C)^2 \ \ldotp
\end{align*}
Da $A$ jedoch eine gewisse Zerfallsbreite $\Gamma_A$ hat, folgt die gemessene invariante Masse $M_{BC}$ einer relativistischen \textsc{Breit-Wigner}-Verteilung mit Normierungsfaktor $a$:
\begin{align*}
N(M_{BC}) = \frac{a}{M_{BC} \left[ \left(M^2_{BC} - M^2_{A}\right)^2 + M^2_{A} \Gamma^2_A\right] } \ \ldotp
\end{align*}
Auch bei Hinzunahme von Untergrundereignissen bei einem idealen Detektor bleibt diese Form erhalten. Da ein realer Detektor jedoch eine endliche Auflösung hat, welches zu \textsc{gauß}verteilten Abweichungen führt, erhält man für einen realen Detektor eine \textsc{Voigt}-Verteilung (Faltung von \textsc{Gauß}- und \textsc{Breit-Wigner}-Verteilung). 
Im Versuch verwenden wir die nicht-relativistische Näherung der \textsc{Breit-Wigner}-Verteilung:
\begin{align*}
N(M_{BC}) = \frac{\frac{\Gamma^2_A}{4}}{\left(M_{BC} - M_{A}\right)^2 + \frac{\Gamma^2_A}{4})} \ \ldotp
\end{align*}
Des Weiteren wird der eigentlich noch vorhandene $\gamma Z^0$-Interferenzterm nicht berücksichtigt. Der Fehler durch diese Approximationen ist vernachlässigbar. 

\subsubsection{Virtuelle Teilchen}
Teilchen können nach der \textsc{Heisenberg}schen Unschärferelation für kurze Zeit die für reelle Teilchen geltende Relation $E^2 - p^2 = m^2$ verletzen. Solche Teilchen bezeichnet man als "`virtuell"' oder als nicht auf der Massenschale liegend. Sie können nicht experimentell gemessen werden, vermitteln aber nach der Theorie die jeweilige Wechselwirkung. Um ein virtuelles Teilchen als solches zu kennzeichnen, verwendet man ein $^*$.

\subsubsection{Feynman-Diagramme}
Mit Hilfe von \textsc{Feynman}-Diagrammen lassen sich quantenfeldtheoretische Prozesse (wie Streuprozesse) graphisch darstellen. Ein- und auslaufenden (reellen) Teilchen weist man hierzu meist durchgezogene Linien zu, internen (virtuellen oder reellen) Teilchen z.\ B.\ gestrichelte, wobei die Zeitachse von links nach rechts läuft. An den Linien angebrachte Pfeile unterscheiden über die Pfeilrichtung bezüglich der Zeitachse zwischen Teilchen- (in Zeitrichtung) und Antiteilchen (gegen Zeitrichtung). Die Knotenpunkte der Reaktionen nennt man Vertices. 

Jedem graphischen Symbol entspricht dabei ein mathematischer Ausdruck. Mit Hilfe der aus dem jeweiligen Lagrangian extrahierbaren \textsc{Feynman}-Regeln lässt sich dann das zum \textsc{Feynman}-Diagramm gehörende Matrixelement aufschreiben.

Beispiele für \textsc{Feynman}-Diagramme sind in im nächsten Abschnitt sowie in \ref{subsec:NuPhysics} aufgeführt.

\subsubsection{Drell-Yan-Prozess}
Protonen sind zusammengesetzte Teilchen. Ihre Konstituenten bezeichnet man dabei als Partonen (z.\ B.\ Quarks, Antiquarks und Gluonen), welche bei den im LHC erreichten Energien als quasi-frei betrachtet werden können (asymptotische Freiheit). Bei den Proton-Proton-Kollisionen im LHC wechselwirken diese Partonen im Wesentlichen inelastisch miteinander. Ein möglicher ablaufender Prozess ist dabei der sogenannte elektromagnetische \textsc{Drell-Yan}-Prozess (siehe Abb. \ref{fig:drellyan}). Anstelle des Photons kann auch ein Eichboson der schwachen Wechselwirkung entstehen.

\subsubsection{Transversalimpuls}
Da die Protonenstrahlen selbst keine Transversalimpulse tragen und Impulserhaltung gilt, sollte die Summe der Transversalimpulse aller an einem Ereignis beteiligten Teilchen stets verschwinden. Mögliche Transversalimpulse der wechselwirkenden Partonen (asymptotische Freiheit) sind auf Grund der hohen Schwerpunktsenergie vernachlässigbar. Sollte die Messung also einen nicht verschwindenden Wert des Transversalimpulses für ein Ereignis ergeben, so lässt sich darauf schließen, dass ein nicht detektiertes Teilchen (z.\ B.\ Neutrino) den fehlenden Transversalimpuls trägt. Es handelt sich also um ein sehr nützliches Konzept.

Bei vernachlässigbaren Teilchenmassen lässt sich folgende Definition des fehlenden Transversalimpulses (auch fehlende Transversalenergie genannt) angeben:
\begin{align}
\displaystyle{\not} E_T \equiv - \sum_k i E_k \sin{\theta_k} \vec{n}_{k,\perp} \ \ldotp
\end{align}
Hierbei bezeichnet $E_k$ die Energie des gemessenen Teilchens $k$, $\theta_k$ den dazugehörigen Streuwinkel und $\vec{n}_{k,\perp}$ den transversalen Richtungsvektor.

\subsection{Der LHC}
Der LHC (Large Hadron Collider) ist kreisförmiger Proton-Proton-Beschleuniger des CERN bei Genf mit einem Durchmesser von $\unit[27]{km}$. Er wurde in den ehemaligen LEP-Tunnel gebaut und sollte ursprünglich 2008 den Betrieb aufnehmen. Nach einigen Startschwierigkeiten befindet er sich zur Zeit (Frühjahr 2010) wieder in der Anlaufphase.

Die aus einer Wasserstoffquelle stammenden Protonen werden nach einer Vorbeschleunigung auf etwa $\unit[450]{GeV}$ in den LHC-Ring injiziert. Dort erfolgt die weitere Beschleunigung auf $\unit[7]{TeV}$ mittels $\unit[400]{MHz}$-Kavitäten. Bei der Kollision der beiden Protonenstrahlen stehen dann $\unit[14]{TeV}$ als Schwerpunktsenergie zur Verfügung. Ferner sind natürlich CMS und Laborsystem identisch. Um Protonen dieser Energie auf der Kreisbahn zu halten sind nahezu ohne Unterbrechung entlang des Strahls große Dipolmagnete mit einer maximalen Feldstärke von $\unit[8.3]{T}$ angebracht. Die Luminosität wird im Normalbetrieb bei $\unit[10^{34}]{s^{-1} cm^{-2}}$ liegen.

An insgesamt vier Punkten können die beiden Protonen-Strahlen zur Kollision gebracht werden. Dort sind neben ATLAS die Experimente ALICE, CMS und LHC-b zu finden.

\subsection{ATLAS}
Da die (simulierten) Daten des ATLAS-Detektor (\textbf{A} \textbf{T}oroidal \textbf{L}HC \textbf{A}pparatu\textbf{s}) Gegenstand dieses Versuches sind, soll hier kurz auf selbigen eingegangen werden. Wie andere Detektoren auch besteht der $\unit[46]{m}$ lange, $\unit[25]{m}$ hohe und $\unit[7000]{t}$ schwere ATLAS-Detektor aus mehreren Komponenten.

In einem Abstand von knapp $\unit[5-10]{cm}$ vom Strahlrohr befindet sich zunächst ein \textbf{Pixeldetektor} aus kleinen Siliziumdetektoren, den einzelnen "`Pixeln"'. Jeder dieser Pixel ist dabei mit einer eigenen Elektronik ausgestattet, um die Datenmenge zu reduzieren, wobei die Daten von bis zu 80 Millionen Pixeln zeitgleich verarbeitet werden können. Dieser Detektor hat die Aufgabe, den Wechselwirkungspunkt möglichst exakt zu rekonstruieren. Ebenfalls zur Spurrekonsstruktion wird der \textbf{SCT} (Silizium-Halbleiter-Tracker), ein  Streifendetektor, verwendet. 

Als nächstes ist ein Übergangsstrahlungsdetektor (\textbf{TRT}) aus dünnen Driftkammern installiert. Mit diesem wird die Übergangsstrahlung ionisierender Teilchen detektiert und hilft so helfen, Elektronen zu identifizieren.

Es schließt sich nun eine \textbf{Magnetspule} an, welche ein Magnetfeld von etwa $\unit[2]{T}$ parallel zum Protonenstrahl bereitstellt. Auf diese Weise bleiben auch kurzlebige Teilchen detektierbar und geladene Teilchen können mittels der Spurkrümmung im Magnetfeld unterschieden werden.

Diese vier Komponenten formen den inneren Detektor. Der äußere Detektor schließt sich an diesen mit dem \textbf{Presampler} an. Dieser detektiert die elektromagnetischen Schauer  und unterscheidet Photonen in Einzelphotonen und Photonenpaare. Dies hilft bei der Auswertung des sich anschließenden \textbf{elektromagnetischen Kalorimeters} (ECAL). Für den vorliegenden Versuch ist er allerdings ohne Relevanz. Im ECAL werden elektromagnetische Schauer erzeugt und detektiert. Im Wesentlichen elektromagnetisch wechselwirkende Teilchen sollen hier möglichst ihre gesamte Energie deponieren. Durch Materie propagierende Elektronen, Positronen und Photonen verlieren ihre Energie dabei v.\ a.\ durch Bremsstrahlung, Paarbildung und \textsc{Compton}streuung. Ein einfallendes Teilchen hat nach einer materialabhängigen Streulänge nur noch den Bruchteil $\frac{1}{e}$ seiner ursprünglichen Energie. Die Ausdehnung eines Schauers wird also von der Energie des den Schauer erzeugenden Teilchens bestimmt.

Als nächstes Element folgt das \textbf{Hadronkalorimeter} (HCAL). Die hierhin gelangenden Hadronen treffen auf die Kerne des Kalorimetermaterials (Eisen) und generieren hadronische Schauer, wenn die Energie der Sekundärteilchen ausreicht, weitere Hadronen zu erzeugen. Deren Anzahl ist dabei erneut charakterisiert durch die Energie des Primärteilchens. Man definiert ferner eine Wechselwirkungslänge $\lambda$, nach welcher das Primärteilchen mit einer Wahrscheinlichkeit von $\frac{1}{e}$ noch nicht stark gewechselwirkt hat. Die Wechselwirkungslänge der starken Wechselwirkung ist dabei deutlich größer als die Streulänge der elekromagnetischen Strahlung. Die Anordnung der beiden Kalorimeter trägt dem Rechnung.

Ganz außen befinden sich schließlich noch die Myonen-Kammern. Da Myonen deutlich schwerer sind als Elektronen haben sie eine sehr hohe Reichweite in Materie. Das Myonen-System nimmt deshalb auch die Hälfte des gesamten ATLAS-Detektors ein und ist mit einem eigenen Magnetsystem ausgestattet.


\subsection{Koordinatensysteme}
Bei der Wahl eines angemessenen Koordinatensystems bieten sich im Wesentlichen drei an. Zum einen besteht die Möglichkeit in einem kartesischen Koordinatensystem die Strahlrichtung als z-Achse zu wählen, zum Mittelpunkt des LHC-Ringes zeigt die x-Achse und orthogonal auf beiden steht die y-Achse.
Zum anderen steht die Wahl eines kanonischen Kugelkoordinatensystems zur Verfügung. Da jedoch die Ereignisse nicht gleichverteilt auftreten, sondern auf Grund der nur geringen Transversalimpulse eine Häufung bei kleinen Winkeln zu erwarten ist, modifiziert man das Koordinatensystem und führt eine neue Größe ein, die Rapidität $y \equiv \frac{1}{2} \ln{\frac{E+p_z}{E-p_z}}$. Rapiditätsdifferenzen sind invariant unter Lorentz-Boosts parallel zu z-Achse. In Systemen in denen die Partonenmassen vernachlässigt werden können erhält man mit dem Winkel $\theta$ zur Strahlachse stattdessen die sogenannte Pseudo-Rapidität
\begin{align}
\eta = - \ln{\left(\tan{\frac{\theta}{2}}\right)}  \ \ldotp
\end{align}
Der ATLAS-Detektor kann Pseudo-Rapiditäten $|\eta| < 2.5$ für Leptonen sowie $|\eta| < 5$ für Hadronen auflösen.

\subsection*{Frage A: Zerfall des $Z^0$-Bosons}
\emph{Welchen Wert hat der Impuls eines Elektrons beim Zerfall des $Z^0$-Bosons ($Z^0 \rightarrow e^+ e^-$) im Ruhesystem des $Z^0$-Bosons?}

Das $Z^0$-Boson wird in Ruhe betrachten, hat also einen verschwindenden Dreier-Impuls $\vec{p}_Z$. Da Vierer-Impuls-Erhaltung gilt, verschwindet auch die Summe der Dreier-Impulse von Positron und Elektron ($\vec{p}_{e^+} = - \vec{p}_{e^-}$). Man betrachte also die 0-Komponente der Vierer-Impulse:

%noch schöner machen:
\begin{eqnarray*}
               E_{Z^0} &=& E_{e^+} + E_{e^-} \\
\Rightarrow  m^2_{Z^0} &=& 2 E_{e^-} \\
                       &=& 2 \sqrt{m^2_e + \vec{p_e}^2} \\
\Rightarrow  \vec{p_e}^2 &=& \frac{m^2_{Z^0}}{4} - m^2_e \\
\Rightarrow  \vec{p_e}   &=& \sqrt{\frac{m^2_{Z^0}}{4} - m^2_e} \\
                         &=& \unit[45.6]{GeV}
\end{eqnarray*}

\subsection*{Frage B: Elektron-Positron-Streuung}
\emph{Wie groß ist der Impuls der $\tau$-Leptonen bei der Reaktion $e^+ e^- \rightarrow \tau^+ \tau^-$ im CMS ($\sqrt{s} = \unit[5]{GeV}$)?}

Es gilt Energieerhaltung ($s = s'$), somit folgt:
\begin{eqnarray*}
    \sqrt{s} &=& E_{\tau^+} + E_{\tau^-} \\
             &=& 2 E_{\tau^+} \\
             &=& 2 \sqrt{m^2_{\tau} + p^2_{\tau}} \\
\Rightarrow  p^2_{\tau} &=& \frac{s}{4} - m^2_{\tau} \\
\Rightarrow  p_{\tau}   &=& \sqrt{\frac{s}{4} - m^2_{\tau}} \\
\Rightarrow  p_{\tau}   &=& \unit[1.76]{GeV}
\end{eqnarray*}

\section{Versuchsdurchführung und Auswertung}
\subsection{Teil 2: Kalibration der Elektronen}\label{subsec:ElektronKalibration}
Die genauste Methode, die Energie eines Elektrons zu bestimmen, ist die Messung der Energie des elektromagnetischen Schauers, den das Elektron im Kalorimeter auslöst. Da das Kalorimeter aus vielen hundert Einzelteilen mit verschiedenen Eigenschaften besteht und  systematische Energieverluste durch inaktive Regionen im Detektor sowie Energieverluste der Elektronen vor Erreichen des Kalorimeters vorliegen, muss diese Energiemessung kalibriert werden. Diese Kalibration lässt sich auf Grund der sehr genau bekannten Eigenschaften des $Z^0$ anhand dessen Zerfall in ein $e^+e^-$-Paar durchführen. Dazu stehen uns Daten solcher gemessener Zerfälle zur Verfügung, die wir mit Hilfe des Programms \texttt{ROOT} analysieren.

Zuerst plotten wir die in Abbildung \ref{fig:el_energy} bzw. Abbildung \ref{fig:pos_energy} dargestellten Histogramme der gemessenen Elektron- bzw. Positron-Energie. Wir stellen fest, dass beide Histogramme einen Peak bei etwa der halben $Z^0$-Masse enthalten. Entsteht bei dem harten Streuprozess ein im Laborsystem ruhendes $Z^0$, so kann es in ein Leptonpaar mit genau entgegengesetzten Impulsen zerfallen. Die beiden Leptonen haben dann jeweils die Energie der halben $Z^0$-Masse, sodass der beschriebene Peak im Histogramm entsteht.

\item nicht genauso viele Anti-Leptonen wie Leptonen vorkommen und

\subsubsection{Neue Physik}
\label{subsubsec:NuPhysics}
Um im ATLAS-Datensatz nach neuer Physik zu suchen, identifizieren wir zuerst die Observablen die sensitiv auf diese sein könnten. Im Falle eines \textbf{Higgs}-Bosons oder zusätzlichen \textbf{schweren Eichbosonen} erwarten wir Ergebnisse analog zum $Z^0$-Paar-Zerfall des Standardmodells. Es sollte sich in beiden Fällen ein neuer Peak in der \textit{invarianten vier-Leptonen-Masse} zeigen. Da keine Neutrinos oder andere lediglich schwach wechselwirkende Teilchen entstehen erwarten wir einen geringen fehlenden Transversalimpuls. Außerdem erwarten wir keine Jets, da keine Quarks oder Gluonen im Endzustand auftreten.

Im Falle von \textbf{SUSY} entstehen dagegen viele lediglich schwach wechselwirkende Teilchen, die im Detektor nicht gemessen werden und so einen hohen \textit{fehlenden Transversalimpuls} verursachen. 

Für das Szenario zusätzlicher \textbf{schwerer Quarks} erwarten wir eine hohe \textit{Anzahl jets} sowie aufgrund der enstehenden Neutrinos einen nicht verschwindenden \textit{fehlenden Transversalimpuls}, der aber aufgrund der geringen Masse der Neutrinos kleiner sein sollte als derjenige im Falle von SUSY.

Aus diesen Überlegungen entwickeln wir die folgenden Schnittkriterien:

\begin{itemize}
\item \textbf{Higgs}: Da wir in Abeschnitt \ref{subsubsec:ZPairs} bereits Hinweise darauf gefunden haben, dass das Higgs-Boson eine kleinere Masse hat als die doppelte $Z^0$ Masse, verwenden wir ein Schnittkriterium, dass alle Ereignisse herausfilter in denen beide $Z^0$ Bosonen reell sind. Außerdem verwenden wir das bereits in Abschnitt \ref{subsubsec:ZPairs} verwendete Schnittkriterium, das alle Ereignisse herausfilert in denen die Summe der transversalen Energie in Kegeln um die Leptonen größer ist als 8 GeV, da das Higgs Boson über ein $Z^0$ Paar zerfällt. 
\item \textbf{SUSY}: Das Histogramm der fehlenden Transversalenergie des $Z^0$ Paar Standardmodell-Datensatzes besitzt den großteil der Einträge im Regime bis ca 100 MeV. Aus diesem Grund setzen wir ein Schnittkriterium, das Ereignisse herausfiltert bei denen die fehlende Transversalenergie kleiner ist als 100 MeV.
\item \textbf{schwere Quarks}: Beim Betrachten der Verteilung der Jet-Anzahl im $Z^0$ Paar Standardmodell-Datensatz stellen wir fest, dass die meisten Einträge eine Jet-Anzahl kleiner als 4 aufweisen. Daher verwenden wir ein Schnittkriterium, das nur Ereignisse zulässt die mehr als 4 Jets aufweisen.
\item \textbf{schwere Eichbosonen}: Da sich die neuen Eichbosonen wie $Z^0$ Bosonen verhalten sollten, verwenden wir die Schnittkriterien aus Abschnitt \ref{subsubsec:ZPairs}.
\end{itemize}

event *atlas_event = new event("ATLASDATA-ZZ-calib.root", "ATLAS");

  if (ev->lep_etiso[0]+ev->lep_etiso[1]+ev->lep_etiso[2]+ev->lep_etiso[3]>8) continue;
  if (ev->lep_id[0]*ev->lep_id[1]*ev->lep_id[2]*ev->lep_id[3] != 11*11*13*13) continue;

  drawer->Binning(20,0., 10.);

  //determine which leptons belong together
  TLorentzVector InvMass1, InvMass2;
  if( ev->lep_id[0] == -ev->lep_id[1] )
  {
    InvMass1 = ev->lep[0]+ev->lep[1];
    InvMass2 = ev->lep[2]+ev->lep[3];
  }
  else if ( ev->lep_id[0] == -ev->lep_id[2] )
  {
    InvMass1 = ev->lep[0]+ev->lep[2];
    InvMass2 = ev->lep[1]+ev->lep[3];
  }
  else if ( ev->lep_id[0] == -ev->lep_id[3] )
  {
    InvMass1 = ev->lep[0]+ev->lep[3];
    InvMass2 = ev->lep[1]+ev->lep[2];
  }

  // determine what kind of Z pair occured
  // zztype =1 both are real, ZZtype=2 both are virtual, ZZtype=3 mixed case
  int zztype;  
  
  zztype = 3; //if nothing is set, mixed case has occured
  if ( fabs(InvMass1.M() - 91.1876) <5 )		//first Z is real
  {
     if ( fabs( InvMass2.M() - 91.1876) <5 )	    //second Z is real
     {
         zztype = 1;
     }
  }
  else											//first Z is virtual
  {
     if ( fabs(InvMass2.M() - 91.1876) < 5 ) 	//second Z is virtual
     {
         zztype = 2;
     }
  }

  //calculate four-lepton-invariant mass
  TLorentzVector l4=ev->lep[0]+ev->lep[1]+ev->lep[2]+ev->lep[3]; 

  //the results are now filled into a histogram, different slots for below and above threshold!
  if ( l4.M() < 2.*91.2 - 6.25) // well below threshold
  {
     drawer->Fill(zztype);
  }
  else if ( l4.M() > 2.*91.2 + 6.25) // well above threshold             
  {
     drawer->Fill(zztype+5); 
  }

}
